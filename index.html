Lucene

  • Originally a sourceforge project
  • Joined Apache Jakarta in Sep 2001
  • Became a top level project in Feb 2005
  • Written by Douglas Cutting
  • Tagline: Lucene is a free (as in free beer) / open source IR library
  • Released under the Apache License

Ports

  • PyLucene
  • Ruby
  • Lucene.Net
  • Delphi
  • Perl
  • C++
  • PHP

Index format

Solr

History

  • Started out as a CNET internal project
  • Joined Apache in 2006
  • Graduated from incubation status in 2007

Important Versions

  • 1.4 in 2009 (Huge Javazone presentation)
  • 3.1 in spring 2011. Joined codebase with Lucene
  • Most recent: November 27th 2011 - 3.5

Solr + Lucene

  • Sharing of features and fixes now happens at a much higher rate

Solr Admin

Solritas

  • Bird's eye view of index
  • Updates when pushing example docs

Java example (SOLRJ)

// Construct the document
SolrInputDocument doc1 = new SolrInputDocument();
doc1.add("id", 1);
doc1.add("price", 10.5);
// Get server
SolrServer server = new CommonsHttpSolrServer("http://localhost:8983/solr/");
server.add(doc1);
server.commit();

Java query example

SolrServer solrServer = getSolrServer();
// construct query
SolrQuery query = new SolrQuery();
query.setQuery("*.*");
query.addSortField("price", SolrQuery.ORDER.asc);
//Get query response
QueryResponse rsp = solrServer.query(query);

//Get documents
SolrDocumentList docs = rsp.getResults();

Ruby query example


solr = RSolr.connect :url => 'http://localhost:8983/solr'
response = solr.get 'select', :params => {
:q => '*:*',
:sort => 'price asc'
}
response["response"]["docs"].each{|doc| puts doc["id"]}

Facet navigation

  • Add facet=true&facet.field=field to the url
  • Will add a facet section to the query response
